About OptiCrop

OptiCrop is an AI-powered crop recommendation system that helps farmers choose the best crop based on soil nutrients and environmental conditions.

Project Objectives

  • Recommend the most suitable crop.
  • Improve agricultural productivity.
  • Support sustainable farming.
  • Optimize resource utilization.

Input Parameters

  • Nitrogen (N)
  • Phosphorus (P)
  • Potassium (K)
  • Temperature
  • Humidity
  • Soil pH
  • Rainfall

Technologies Used

Python Flask Scikit-learn Pandas NumPy HTML5 CSS3 Machine Learning
